# Announcing Consiglio

By Rick Goldberg, Aniviza LLC Productions

Consiglio is an open-source desktop workspace for Codex tasks that need to survive beyond one terminal session.

It starts ready to type, reconnects the last task automatically, and preserves conversations, drafts, repositories, provider settings, and Codex thread IDs across restarts and crashes. While Codex works, Consiglio shows streaming activity, commands, approvals, and errors instead of leaving the user guessing.

The app also includes a workspace file browser with text and image previews, multiple persistent tasks, encrypted API-key storage for provider and MCP workflows, and support for Codex, Ollama, remote llama.cpp, and discovered LAN providers.

Version 0.1.0 is available now for Linux, Windows, and macOS. Linux is the primary platform.
