# Consiglio 0.1.0

The first public release of Consiglio, an open-source desktop workspace for Codex tasks that outlive the terminal.

## Highlights

- Resume the last task automatically after restarting the app.
- Preserve conversations, drafts, repository context, providers, and Codex thread IDs.
- See visible working state, streamed responses, commands, approvals, and copyable errors.
- Browse workspace files and preview text and images beside the conversation.
- Keep multiple persistent tasks without repeatedly choosing a workspace.
- Store provider and MCP API keys with Electron `safeStorage` encryption.
- Use Codex, Ollama, remote llama.cpp, and discovered LAN providers.

## Downloads

- **Linux:** Debian package or AppImage. Linux is the primary platform.
- **Windows:** installer or portable executable.
- **macOS:** DMG or ZIP for Intel Macs.

A working Codex CLI installation is required. Use `SHA256SUMS.txt` to verify downloaded packages.
